# Задача 1: Таблица синусов

Заполнить масив float/double значениями синуса (один период на всю длину массива), количество элементов 10^7. 
Посчитать сумму и вывести в терминал.

## Сборка проекта

```bash
cmake -S. -Bbuild -DUSE_DOUBLE=OFF
cd build
make
./myexec
```

## Выбор типа массива

Тип элементов массива (`float` или `double`) выбирается через CMake-опцию `USE_DOUBLE`:

| Тип массива | Команда конфигурации                     |
|-------------|------------------------------------------|
| `float`     | `cmake -S. -Bbuild -DUSE_DOUBLE=OFF`     |
| `double`    | `cmake -S. -Bbuild -DUSE_DOUBLE=ON`      |



## Результаты выполнения

| Тип массива | Сумма элементов | Команда сборки                              |
|-------------|-----------------|---------------------------------------------|
| `float`     | `-5.55091e-05`  | `cmake -S. -Bbuild -DUSE_DOUBLE=OFF && make` |
| `double`    | `4.89582e-11`   | `cmake -S. -Bbuild -DUSE_DOUBLE=ON && make`  |
